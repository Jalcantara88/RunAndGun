DESCRIPTION

Side Scrolling Action Platformer Infinite Runner!
See how far you can go while dodgine bullets and jumping to the next platform.
But watch out, the enemies are always spawning and the game scrolls faster and faster!
You only have 3 lives. So get ready to RUN & GUN

ABOUT THE BUILD

This was a project for MintBean.io's Build An Action Game Hack-a-thon / Learn-a-thon! In this, you only get less than one week to build an action game.

This was my first time using Phaser.JS for anything. The documentation can be a bit hard to get through, but I was able to get through it with online research. I have worked on game design befor, but only as an Artist. So I had some understanding of the process. This project really helped me hone in on organizing my code and thinking differently to problem solve. I am quite happy with how it turned out and will probably use Phaser again.

The project was built using Parcel, this page was styled using Bootstrap and deployed onto Netlify!
